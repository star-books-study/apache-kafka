# Chapter 5. 카프카 실전 프로젝트
## 5.1. 웹 페이지 이벤트 적재 파이프라인 생성
- 카프카는 웹 이벤트 수집 파이프라인을 만들고 운영하기에 최적화되어 있다

### 5.1.1. 요구사항
- 하둡 : 대용량 데이터 분석처리
  - HDFS : 대용량 파일을 하둡에 안정적으로 저장할 수 있게 하는 파일 시스템

### 5.1.2. 정책 및 기능 정의

#### 적재 정책
- 카프카는 멱등성 프로듀서를 통해 정확히 한번 전달을 지원한다
  - 이 때 전달은, 
  - **프로듀서 ~ 브로커까지 전달**되는 것을 의미한다
